Call GPT-4 and Claude from Python — send messages, stream responses, manage conversation history, and control output.

Calling LLMs from Python

The OpenAI and Anthropic APIs share a similar architecture: you send a messages array to a model endpoint and get a completion back. Both charge per token. Both support system prompts, multi-turn conversations, and streaming. Learning one makes the other trivial.

Mental model: You are not "talking to AI" — you are calling a function that takes text in and returns text out. The skill is crafting the right input to get the right output.

OpenAI API

pip install openai
from openai import OpenAI
import os

client = OpenAI(api_key=os.environ["OPENAI_API_KEY"])

# Single message
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {"role": "system", "content": "You are a helpful Python tutor."},
        {"role": "user",   "content": "Explain list comprehensions in 2 sentences."}
    ],
    temperature=0.7,
    max_tokens=200
)

print(response.choices[0].message.content)
print("Tokens used:", response.usage.total_tokens)

Claude (Anthropic) API

pip install anthropic
import anthropic, os

client = anthropic.Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

message = client.messages.create(
    model="claude-opus-4-5",
    max_tokens=1024,
    system="You are an expert Python for AI instructor.",
    messages=[
        {"role": "user", "content": "What is the difference between a list and a tuple?"}
    ]
)

print(message.content[0].text)
print("Input tokens:", message.usage.input_tokens)
print("Output tokens:", message.usage.output_tokens)

Multi-turn Conversations

from openai import OpenAI
client = OpenAI(api_key=os.environ["OPENAI_API_KEY"])

history = [{"role": "system", "content": "You are a Python tutor."}]

while True:
    user_input = input("You: ").strip()
    if user_input.lower() in ("quit", "exit"): break
    history.append({"role": "user", "content": user_input})
    response = client.chat.completions.create(
        model="gpt-4o-mini", messages=history
    )
    reply = response.choices[0].message.content
    history.append({"role": "assistant", "content": reply})
    print(f"AI: {reply}\n")

Streaming Responses

# Stream tokens as they arrive -- better UX for long outputs
stream = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Explain RAG in detail."}],
    stream=True
)
for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)
print()  # newline at end

Lab: Build an AI Code Reviewer

Use the Claude or OpenAI API to build a tool that reviews Python code and suggests improvements.

1

Set up your API key

In Colab, click the key icon (Secrets) on the left sidebar. Add ANTHROPIC_API_KEY with your key. Then:

from google.colab import userdata
import anthropic

api_key = userdata.get("ANTHROPIC_API_KEY")
client = anthropic.Anthropic(api_key=api_key)
print("Client ready!")
2

Write the code reviewer function

def review_code(code: str) -> str:
    message = client.messages.create(
        model="claude-opus-4-5",
        max_tokens=1024,
        system="You are a senior Python engineer. Review code for: bugs, style issues, performance problems, and security risks. Be specific and constructive.",
        messages=[{"role": "user", "content": f"Review this Python code:\n\n```python\n{code}\n```"}]
    )
    return message.content[0].text

print("Function defined!")
3

Test it on some bad code

bad_code = """
def get_user(id):
    import sqlite3
    conn = sqlite3.connect("db.sqlite")
    cursor = conn.cursor()
    cursor.execute(f"SELECT * FROM users WHERE id={id}")
    return cursor.fetchone()
"""

feedback = review_code(bad_code)
print(feedback)

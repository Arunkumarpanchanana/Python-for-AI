Data Visualization

Use matplotlib and seaborn to explore datasets visually — distributions, correlations, training curves, and dashboards.

30 min Intermediate Hands-on Lab 5 Quiz Questions

Why Visualize Data?

Before building any model, you must see your data. Patterns that take hours to spot in raw numbers are obvious in a chart. Outliers, class imbalances, correlations, distributions — all visible in seconds with the right plot.

Python has two primary visualization layers: Matplotlib (the foundation) and Seaborn (a high-level wrapper that produces publication-quality charts with less code). For AI work, knowing both is essential.

Matplotlib Basics

import matplotlib.pyplot as plt
import numpy as np

# Line plot
x = np.linspace(0, 10, 100)
plt.figure(figsize=(8, 4))
plt.plot(x, np.sin(x), label="sin(x)", color="steelblue")
plt.plot(x, np.cos(x), label="cos(x)", color="orange", linestyle="--")
plt.xlabel("x"); plt.ylabel("y")
plt.title("Sin and Cos")
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

Seaborn for Statistical Plots

import seaborn as sns
import pandas as pd

# Distribution plot
sns.histplot(df["age"], bins=30, kde=True)

# Box plot -- shows quartiles, outliers
sns.boxplot(x="department", y="salary", data=df)

# Correlation heatmap -- critical before ML
corr = df.select_dtypes("number").corr()
sns.heatmap(corr, annot=True, fmt=".2f", cmap="coolwarm")
plt.title("Feature Correlations")
plt.show()

# Pairplot -- scatter matrix for multiple features
sns.pairplot(df[["age", "salary", "experience"]], hue="department")

Training Curves (Essential for AI)

# Visualising model training -- every AI practitioner needs this
train_loss = [0.9, 0.7, 0.55, 0.42, 0.33, 0.27, 0.22, 0.19]
val_loss   = [0.95, 0.72, 0.60, 0.50, 0.48, 0.47, 0.49, 0.52]
epochs = range(1, len(train_loss) + 1)

plt.figure(figsize=(8, 4))
plt.plot(epochs, train_loss, label="Train Loss", marker="o")
plt.plot(epochs, val_loss,   label="Val Loss",   marker="s", linestyle="--")
plt.axvline(x=6, color="red", alpha=0.4, label="Overfitting starts")
plt.xlabel("Epoch"); plt.ylabel("Loss")
plt.title("Training vs Validation Loss")
plt.legend(); plt.show()
Always plot loss curves. If validation loss starts rising while training loss keeps falling, your model is overfitting. Visualising this early saves hours of wasted training.

Subplots for Dashboards

fig, axes = plt.subplots(1, 3, figsize=(15, 4))

axes[0].hist(df["age"], bins=20, color="steelblue")
axes[0].set_title("Age Distribution")

axes[1].scatter(df["age"], df["salary"], alpha=0.5)
axes[1].set_title("Age vs Salary")

axes[2].bar(df["dept"].value_counts().index, df["dept"].value_counts().values)
axes[2].set_title("Headcount by Department")

plt.tight_layout()
plt.show()

Load and prepare

import pandas as pd, matplotlib.pyplot as plt, seaborn as sns
url = "https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv"
df = pd.read_csv(url)

Survival rate by class and sex

plt.figure(figsize=(10, 4))

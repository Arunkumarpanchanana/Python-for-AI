What Is NumPy and Why Does It Matter?

NumPy (Numerical Python) is the foundation of the entire scientific Python stack. Every major AI library — PyTorch, TensorFlow, scikit-learn, pandas — builds on top of NumPy arrays. Understanding NumPy means understanding how data flows through AI systems.

The key insight: a NumPy array is not a Python list. It stores all elements in contiguous memory as a single C array, which makes operations 10–100x faster than equivalent Python loops. When you multiply two NumPy arrays, the math runs in compiled C — Python just orchestrates it.

The core rule: Avoid Python loops over array elements. Instead, use NumPy's built-in vectorized operations — they run in C and are dramatically faster.

Creating Arrays

import numpy as np

# From a list
a = np.array([1, 2, 3, 4, 5])
print(a.dtype)   # int64
print(a.shape)   # (5,)

# 2D array (matrix)
matrix = np.array([[1, 2, 3], [4, 5, 6]])
print(matrix.shape)  # (2, 3) -- 2 rows, 3 cols

# Special arrays
np.zeros((3, 4))        # 3x4 matrix of zeros
np.ones((2, 5))         # 2x5 matrix of ones
np.arange(0, 10, 2)     # [0, 2, 4, 6, 8]
np.linspace(0, 1, 5)    # [0.0, 0.25, 0.5, 0.75, 1.0]
np.random.rand(3, 3)    # 3x3 uniform random [0,1)
np.random.randn(100)    # 100 samples from standard normal

Vectorized Operations

a = np.array([1, 2, 3, 4])
b = np.array([10, 20, 30, 40])

# Element-wise arithmetic -- no loops needed
print(a + b)    # [11 22 33 44]
print(a * b)    # [10 40 90 160]
print(a ** 2)   # [ 1  4  9 16]
print(np.sqrt(b))  # [3.16 4.47 5.48 6.32]

# Aggregations
print(a.sum())    # 10
print(a.mean())   # 2.5
print(a.std())    # standard deviation
print(a.max(), a.min(), a.argmax())  # max, min, index of max

# Boolean indexing -- essential for AI data filtering
data = np.array([15, 8, 23, 4, 42, 7])
print(data[data > 10])  # [15 23 42]

Shapes, Reshaping, and Broadcasting

# Reshape -- change dimensions without changing data
flat = np.arange(12)         # [0 1 2 ... 11]
grid = flat.reshape(3, 4)    # 3x4 matrix
cube = flat.reshape(2, 2, 3) # 3D tensor

# Broadcasting -- NumPy stretches smaller arrays to match larger
matrix = np.ones((3, 4))
row    = np.array([1, 2, 3, 4])  # shape (4,)
result = matrix + row            # row added to each of 3 rows
print(result.shape)  # (3, 4)

# Axis operations -- critical for ML
m = np.array([[1,2,3],[4,5,6]])
print(m.sum(axis=0))  # sum down columns: [5 7 9]
print(m.sum(axis=1))  # sum across rows:  [6 15]

Linear Algebra (the Heart of ML)

# Dot product and matrix multiplication
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
print(np.dot(a, b))  # 32  (1*4 + 2*5 + 3*6)

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
print(A @ B)   # matrix multiply: [[19 22],[43 50]]

# Transpose
print(A.T)     # [[1 3],[2 4]]

# Determinant and inverse (from linalg submodule)
print(np.linalg.det(A))   # -2.0
print(np.linalg.inv(A))   # inverse matrix
Why this matters: Neural networks are matrix multiplications. Every forward pass through a layer is output = inputs @ weights + bias. NumPy makes this readable and fast.

Lab: NumPy for AI Data

Work with arrays, perform vectorized math, and simulate a neural network layer calculation.

1

Install and import NumPy

In a new Colab notebook:

import numpy as np
print("NumPy version:", np.__version__)
2

Create and inspect arrays

# Simulate a dataset: 100 samples, 5 features each
data = np.random.randn(100, 5)
print("Shape:", data.shape)
print("Mean per feature:", data.mean(axis=0).round(3))
print("Std per feature:", data.std(axis=0).round(3))
print("Min:", data.min(), "Max:", data.max())
3

Normalize the data (z-score)

Normalisation is essential before feeding data into ML models.

mean = data.mean(axis=0)
std  = data.std(axis=0)
normalized = (data - mean) / std
print("After normalization:")
print("Mean:", normalized.mean(axis=0).round(3))  # ~[0,0,0,0,0]
print("Std: ", normalized.std(axis=0).round(3))   # ~[1,1,1,1,1]
4

Simulate a single neural network layer

A dense layer: output = ReLU(inputs @ weights + bias)

inputs  = normalized[:10]          # 10 samples, 5 features
weights = np.random.randn(5, 3)    # 5 inputs -> 3 neurons
bias    = np.zeros(3)

raw    = inputs @ weights + bias   # shape (10, 3)
output = np.maximum(0, raw)        # ReLU activation
print("Layer output shape:", output.shape)  # (10, 3)
print("Sample output (first row):", output[0].round(3))

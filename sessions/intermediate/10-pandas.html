Pandas: Data Analysis

Load, clean, filter, and aggregate real datasets using pandas — the essential data toolkit for every AI project.

40 min Intermediate Hands-on Lab 5 Quiz Questions

Why Pandas?

Before you can train any AI model, you need to load, clean, and explore your data. Pandas is Python's go-to library for that work. It provides two core data structures: Series (a labelled 1D array) and DataFrame (a labelled 2D table — think spreadsheet with superpowers).

Real-world rule: 80% of data science work is cleaning data. Pandas makes that painful reality manageable.

Loading and Inspecting Data

import pandas as pd

# Load a CSV
df = pd.read_csv("data.csv")

# Essential first look
print(df.shape)          # (rows, columns)
print(df.head())         # first 5 rows
print(df.tail(3))        # last 3 rows
print(df.info())         # dtypes, null counts
print(df.describe())     # count, mean, std, min, quartiles, max
print(df.columns.tolist())
print(df.isnull().sum()) # nulls per column

Selecting and Filtering

# Select one column (returns Series)
ages = df["age"]

# Select multiple columns (returns DataFrame)
subset = df[["name", "age", "salary"]]

# Filter rows
seniors     = df[df["age"] > 60]
high_earners = df[df["salary"] > 100_000]
both        = df[(df["age"] > 30) & (df["salary"] > 50_000)]

# iloc: position-based   loc: label-based
df.iloc[0]          # first row
df.iloc[0:5, 1:3]   # rows 0-4, columns 1-2
df.loc[0, "name"]   # row index 0, column "name"

Cleaning Data

# Handle missing values
df.dropna()                       # drop any row with a null
df.fillna(0)                      # fill nulls with 0
df["age"].fillna(df["age"].mean()) # fill with column mean

# Rename columns
df.rename(columns={"old_name": "new_name"}, inplace=True)

# Convert types
df["price"] = df["price"].astype(float)

# Remove duplicates
df.drop_duplicates(inplace=True)

# Apply a function to a column
df["name_upper"] = df["name"].apply(str.upper)
df["category"] = df["score"].apply(lambda x: "high" if x > 75 else "low")

Grouping and Aggregating

# groupby: split-apply-combine
summary = df.groupby("department")["salary"].mean()

# Multiple aggregations
stats = df.groupby("department").agg(
    avg_salary=("salary", "mean"),
    headcount=("name", "count"),
    max_salary=("salary", "max")
)

# Sort and limit
top5 = df.sort_values("salary", ascending=False).head(5)

# Value counts -- great for categorical analysis
df["department"].value_counts()

Saving Results

df.to_csv("cleaned.csv", index=False)
df.to_json("data.json", orient="records")
df.to_excel("report.xlsx", index=False)  # pip install openpyxl

Clean and engineer features

df["Age"].fillna(df["Age"].median(), inplace=True)

Extract insights

survival_by_class = df.groupby("Pclass")["Survived"].mean()

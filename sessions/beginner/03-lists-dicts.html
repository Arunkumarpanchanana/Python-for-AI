Lists, Tuples & Dictionaries

The data structures behind every AI project — from datasets to API responses.

30 min Beginner Hands-on Lab

Lists

fruits = ["apple", "banana", "cherry"]
print(fruits[0])    # "apple"
print(fruits[-1])   # "cherry"
print(fruits[1:3])  # ["banana","cherry"]
fruits.append("date"); fruits.remove("banana")
print(len(fruits)); print(sorted(fruits))

List Comprehensions

nums = list(range(1,11))
squares    = [x**2 for x in nums]
evens      = [x for x in nums if x % 2 == 0]
normalized = [x/max(nums) for x in nums]  # ML pattern!

Tuples

point = (3, 4)
x, y = point   # unpacking
# Tuples can be dict keys; lists cannot
grid = {(0,0): "start", (1,0): "wall"}

Dictionaries

student = {"name":"Alice","grade":95}
print(student["name"])
print(student.get("age", 0))  # 0 if missing
student["grade"] = 98
for k, v in student.items():
    print(f"{k}: {v}")

List of Dicts — Real Data Pattern

employees = [
    {"name":"Alice","role":"engineer","salary":120000},
    {"name":"Bob",  "role":"designer","salary":95000},
]
engineers = [e for e in employees if e["role"]=="engineer"]
avg = sum(e["salary"] for e in employees)/len(employees)
print(f"Avg: ${avg:,.0f}")

Lab: Student Grade Tracker

Build a grade system using lists and dicts.

1

Create data

students = [
    {"name":"Alice","grades":[88,92,95,78,90]},
    {"name":"Bob",  "grades":[72,68,75,80,71]},
    {"name":"Carol","grades":[95,98,92,97,99]},
]
2

Calculate averages

for s in students:
    s["avg"] = round(sum(s["grades"])/len(s["grades"]),1)
    print(f"{s['name']}: {s['avg']}")
3

Top student

top = max(students, key=lambda s: s["avg"])
above80 = [s["name"] for s in students if s["avg"] >= 80]
print(f"Top: {top['name']} ({top['avg']})")
print(f"Above 80: {above80}")

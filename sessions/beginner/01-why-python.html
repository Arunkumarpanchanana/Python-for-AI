Python Dominates AI — Here's Why

If you look at every major AI library — TensorFlow, PyTorch, scikit-learn, Hugging Face, LangChain, the OpenAI SDK — they're all Python-first. This isn't a coincidence. Python won the AI race for three reasons: readable syntax that reads like plain English, a massive ecosystem of scientific libraries, and a community that rallied around it early.

When Google released TensorFlow in 2015 and Facebook released PyTorch in 2016, both chose Python as the primary interface. That decision locked in Python's dominance. Today, over 75% of all machine learning repositories on GitHub are primarily Python.

The real reason Python won: Researchers are not professional software engineers. They needed a language fast to write, easy to read, and forgiving of imperfect code. Python delivered all three. Speed of experimentation beats speed of execution in research.

Setting Up: Google Colab (Recommended)

For this course, we recommend Google Colab — a free, browser-based Python environment that requires zero installation. You get a Jupyter notebook with a GPU, access to all major packages, and your work is saved to Google Drive automatically.

Go to Colab: Visit colab.research.google.com and sign in with your Google account. Click "New notebook". That's it — you're ready to code.

Your First Python Code

Let's write real Python right now. This is all you need to get started:

# This is a comment — Python ignores it
print("Hello, AI world!")

# Basic math
print(2 + 2)        # 4
print(10 / 3)       # 3.333...
print(10 // 3)      # 3 (integer division)
print(2 ** 8)       # 256 (2 to the power of 8)

# Variables
name = "Arunkumar"
year = 2026
print(f"Welcome, {name}! It's {year}.")

# Check your Python version
import sys
print(sys.version)
Python 3 only: This course uses Python 3.10+. If you're on an old machine with Python 2, use Google Colab — it always runs the latest Python 3.

Lab: Your First Python Session

Open Google Colab and run your first Python code. No install needed.

1

Open Google Colab

Go to colab.research.google.com, sign in with Google, and click New notebook. You'll see a cell with a ▶ button on the left.

2

Run your first print

Click in the cell, type the following, then press Shift+Enter to run:

print("Hello, Python for AI!")
print(2 ** 10)  # Should output 1024

You should see output below the cell immediately.

3

Create variables and an f-string

Add a new cell (click + Code) and run:

name = "Your Name"  # Replace with your actual name
skill = "Python for AI"
sessions = 22

print(f"Hi, I'm {name}!")
print(f"I'm learning {skill} across {sessions} sessions.")
4

Install a package

In Colab, you can install packages with ! prefix. Run:

!pip install anthropic -q
print("Package installed!")

This shows how easy it is to add any library to your environment.

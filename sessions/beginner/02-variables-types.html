Variables and Assignment

Python is dynamically typed — no type declarations needed.

x = 42
pi = 3.14159
name = "Alice"
active = True
nothing = None
Convention: Use snake_case.

Core Data Types

Type Checking and Conversion

print(type(42))        # <class 'int'>
print(int("100"))      # 100
print(float("3.14"))   # 3.14
print(str(42))         # "42"
print(bool(0))         # False
print(bool("hello"))   # True

String Operations

s = "  Hello, World!  "
print(s.strip())           # "Hello, World!"
print(s.lower())
print(s.replace("o","0"))
print(s.split(","))        # list

name = "Alice"; score = 95.5
print(f"Student: {name}, Score: {score:.1f}%")

Operators

print(10 / 3)   # 3.333
print(10 // 3)  # 3 floor
print(10 % 3)   # 1 remainder
print(2 ** 10)  # 1024
print(5 > 3)    # True
print(True and False)  # False

Lab: Temperature Converter

Build a type-aware converter using variables and f-strings.

1

Celsius to Fahrenheit

celsius = 100.0
fahrenheit = (celsius * 9/5) + 32
print(f"{celsius}C = {fahrenheit}F")
2

Personal info card

name = "Your Name"; age = 25
print(f"Name: {name} | Age: {age}")
3

Type inspector

for v in [42, 3.14, "hello", True, None]:
    print(f"{str(v):<10} type={type(v).__name__}")

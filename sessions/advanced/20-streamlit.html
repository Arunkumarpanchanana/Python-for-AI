Build AI Web Apps with Streamlit

Build and deploy interactive AI web apps in pure Python — no HTML or CSS needed. From script to public URL in minutes.

40 min Advanced Hands-on Lab 5 Quiz Questions

Why Streamlit?

Streamlit lets you build interactive AI web apps in pure Python — no HTML, CSS, or JavaScript required. A 50-line Python script becomes a polished web interface. It is the fastest way to demo an AI prototype to stakeholders or ship an internal tool.

pip install streamlit anthropic

Your First Streamlit AI App

# Save as app.py, then run: streamlit run app.py
import streamlit as st
from anthropic import Anthropic

st.title("AI Text Tools")
st.write("Powered by Claude")

client = Anthropic(api_key=st.secrets["ANTHROPIC_API_KEY"])

task = st.selectbox("Choose a task:", ["Summarise", "Translate", "Explain Like I'm 5"])
text = st.text_area("Enter your text:", height=150)

if st.button("Run") and text:
    with st.spinner("Thinking..."):
        prompt = f"{task} the following text:\n\n{text}"
        msg = client.messages.create(
            model="claude-opus-4-5", max_tokens=500,
            messages=[{"role":"user","content":prompt}]
        )
        st.success("Done!")
        st.write(msg.content[0].text)

Building a Chat Interface

import streamlit as st
from anthropic import Anthropic

st.title("PythonBot")
client = Anthropic(api_key=st.secrets["ANTHROPIC_API_KEY"])

# Session state persists across reruns within a session
if "history" not in st.session_state:
    st.session_state.history = []

# Display chat messages
for msg in st.session_state.history:
    with st.chat_message(msg["role"]):
        st.write(msg["content"])

# Chat input
if prompt := st.chat_input("Ask me anything about Python..."):
    st.session_state.history.append({"role":"user","content":prompt})
    with st.chat_message("user"):
        st.write(prompt)
    with st.chat_message("assistant"):
        with st.spinner("Thinking..."):
            r = client.messages.create(
                model="claude-opus-4-5", max_tokens=512,
                system="You are a Python for AI expert.",
                messages=st.session_state.history
            )
            reply = r.content[0].text
            st.write(reply)
    st.session_state.history.append({"role":"assistant","content":reply})

Deploying to Streamlit Cloud

1. Push your app to a public GitHub repository.
2. Go to share.streamlit.io and sign in with GitHub.
3. Click "New app", select your repo and app.py.
4. Add your API key under Settings → Secrets: ANTHROPIC_API_KEY = "sk-ant-..."
5. Deploy — your app gets a public URL in ~60 seconds.

Lab: Build an AI Writing Assistant

Build a Streamlit app that helps users write, rewrite, and improve text using Claude.

1

Create the app

Install Streamlit and create writing_app.py:

%%writefile writing_app.py
import streamlit as st
import anthropic, os

st.set_page_config(page_title="AI Writing Assistant", page_icon="+")
st.title("AI Writing Assistant")

api_key = os.environ.get("ANTHROPIC_API_KEY","")
client = anthropic.Anthropic(api_key=api_key)

task = st.selectbox("Task", [
    "Make it more professional",
    "Make it simpler (ELI5)",
    "Fix grammar and spelling",
    "Make it shorter",
    "Make it more engaging",
])

text = st.text_area("Your text:", height=200, placeholder="Paste your text here...")

if st.button("Improve Text") and text.strip():
    with st.spinner("Improving..."):
        r = client.messages.create(
            model="claude-opus-4-5", max_tokens=600,
            messages=[{"role":"user","content":f"{task}:\n\n{text}"}]
        )
        st.subheader("Improved Version:")
        st.write(r.content[0].text)
        st.download_button("Download", r.content[0].text, "improved.txt")
2

Run the app

In your terminal (not Colab):

pip install streamlit anthropic
ANTHROPIC_API_KEY=your_key streamlit run writing_app.py

Your browser opens automatically at http://localhost:8501

3

Add a word count metric

Add these lines before the text area in your app:

col1, col2, col3 = st.columns(3)
if text:
    col1.metric("Words", len(text.split()))
    col2.metric("Characters", len(text))
    col3.metric("Est. tokens", len(text) // 4)
4

Deploy to Streamlit Cloud

Push to GitHub, then go to share.streamlit.io. Connect your repo, add your API key as a secret, and deploy. Share the public URL with anyone!
